# kigstudio

基于体素的 3D 打印模型处理工具，支持 STL 体素化、碰撞编辑、智能分割与标记体素刷选。

## 功能特性

### 核心处理
- **STL 体素化**：加载 STL 模型并转换为体素网格，支持自定义体素大小
- **VXGrid 导入/导出**：独立的二进制体素文件格式（zlib 压缩），支持跨项目复用体素数据
- **项目系统**：完整的项目保存/加载（JSON 元数据 + 二进制体素数据），支持 STL 自动恢复

### 碰撞与分割
- **物体编辑器**：双 Tab 设计
  - **碰撞编辑**：支持 5 种分割模式
    - 碰撞体分割（基于碰撞组）
    - 平面分割（自定义空间平面）
    - 凹锥分割（Concave Cone）
    - 断连通块分割（Split Disconnected）
    - BFS 邻近分割（基于标记体素进行广度优先扩散）
  - **体素选择**：表面体素刷选、标记体素管理
- **自动/手动更新**：支持自动传播分割结果到子节点，也可手动控制

### 体素标记与刷选
- **表面体素缓存**：自动计算模型表面体素，支持后台线程初始化
- **笔刷刷选**：左键添加 / Shift+擦除标记体素，支持笔刷范围调节
- **独立历史记录**：标记体素拥有独立的撤销/重做栈（50 步），与碰撞编辑历史互不干扰
- **标记体素持久化**：保存/加载为独立的 `.vxgrid` 文件

### 渲染与交互
- **双模式渲染**：Mesh 模式与 Voxel 模式可切换显示
- **延迟渲染管线**：基于 bgfx 的 G-Buffer 延迟渲染
- **体素高亮浮起**：标记体素以固定世界距离浮起显示，无视距穿模问题
- **导航节点图**：可视化体素分割的树状结构，支持缩略图
- **鼠标世界坐标拾取**：支持 3D 场景中的精确点击拾取

### 导入导出
- **导出 ASCII STL**：将当前节点体素导出为 STL 文件
- **批量导出**：一键导出所有叶子节点到 `exported_stl/` 目录
- **重新加载 STL**：支持修改体素大小后重新体素化

## 技术栈

| 组件 | 说明 |
|------|------|
| C++20 | 核心语言标准 |
| CMake | 构建系统 |
| SDL2 | 窗口与输入 |
| bgfx | 跨平台图形渲染 |
| Dear ImGui / ImNodes | 即时模式 UI 与节点编辑器 |
| tinyfiledialogs | 系统文件对话框 |
| cJSON | 项目元数据序列化 |
| zlib | 体素数据压缩 |

## 构建

```bash
mkdir build && cd build
cmake ..
cmake --build . --config Debug
```

> **Windows 注意**：`std::ofstream`/`std::ifstream` 的 `const char*` 构造函数使用 ANSI 编码，项目中已使用宽字符路径修复 UTF-8 支持。确保 `zd.dll` (zlib Debug) 在可执行文件同级目录。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `O` | 打开 STL 文件 |
| `Ctrl+O` | 加载项目 |
| `Ctrl+S` | 保存项目 |
| `Ctrl+Shift+S` | 另存为 |
| `Ctrl+Z` | 撤销（根据当前 Tab 路由到碰撞编辑或标记体素历史） |
| `Ctrl+Y` | 重做 |
